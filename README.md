# pyATS Gemini-CLI Extension
An Extension for Gemini-CLI that enables pyATS 

## How to add to Gemini-CLI (right now)

```bash
mkdir -p ~/.gemini/extensions

git clone https://github.com/automateyournetwork/pyATS_GeminiCLI_Extension.git \
    ~/.gemini/extensions/pyATS
```
